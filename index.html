Hello, world!

Bootstrap (anteriormente Twitter Bootstrap ) es un marco CSS gratuito y de código abierto orientado al desarrollo de sitios web front-end responsivos y con prioridad para dispositivos móviles . Contiene plantillas de diseño basadas en HTML , CSS y (opcionalmente) JavaScript para tipografía , formularios , botones , navegación y otros componentes de interfaz. A partir de mayo de 2023 , Bootstrap es el 17.º proyecto con más estrellas (la 4.ª biblioteca con más estrellas) en GitHub , con más de 164 000 estrellas. Según W3Techs, el 19,2 % de todos los sitios web utilizan Bootstrap.
